# B. HARRISRAJ PORTFOLIO

A high-end, cinematic "Scrollytelling" personal portfolio built with **Next.js 14**, **Framer Motion**, and **Tailwind CSS**. This website features an image-sequence-driven scroll animation that blends seamlessly into a modern, dark-themed responsive UI.

## 🚀 Experience the Live Portfolio
Visit my professional showcase at: [Portfolio Link] (or local development at `http://localhost:3000`)

## ✨ Key Features
- **Scrollytelling Hero**: A high-performance HTML5 Canvas-based image sequence that scrubs as you scroll.
- **Modern UI/UX**: Professional glassmorphic design system using Vanilla CSS and Tailwind.
- **Project Case Studies**: Highlights key projects like **VOTECHAIN** (Blockchain) and **CAMPUS HUB** (Next.js/Node.js).
- **Personal Identity**: Integrated educational background (Meenakshi Sundararajan Engineering College) and creative pursuits (Portrait Artist/Guitarist).
- **Real-world Communication**: Fully functional "Quick Message" feature connected to **Formspree** for direct client outreach.

## 🛠️ Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Form Handling**: [Formspree](https://formspree.io/)

## 📂 Project Structure
- `/app`: Next.js App Router pages and global styles.
- `/components`: Reusable UI components (Navbar, Hero, About, Skills, Projects, Art, Contact).
- `/public`: Static assets including sequence frames and profile images.

## 🛠️ Getting Started
1. **Clone the repository**:
   ```bash
   git clone https://github.com/harrisraj1421/portfolio-v2.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Configuration**: 
   Update your Formspree ID in `components/ContactModal.tsx` to receive real messages.
